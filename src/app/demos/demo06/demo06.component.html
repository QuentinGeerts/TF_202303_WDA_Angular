Demo 06 - Les Pipes

Les Pipes (ou filtres) sont des fonctions qui permettent de transformer les données affichées dans la vue. Les pipes sont utilisés pour formater, trier, filtrer ou limiter les données affichées dans la vue.

Les pipes sont utilisés dans les expressions d'interpolation ({{ " {{ expression | pipe }}" }}) ou dans les directives de liaison de propriété ([prop]="expression | pipe").

Angular fournit un certains nombre de pipes intégrés, tels que : "date", "lowercase", "uppercase", "titlecase", "currency", "decimal", "percent", "async", "json", etc.

Il est également possible de créer des pipes personnalisés pour répondre aux besoins spécifiques de l'application.

Exemples :

Chaine de caractères

UpperCasePipe

LowerCasePipe

TitleCasePipe

Avant : {{ maVariable1 }}

{{ maVariable1 | uppercase }}

{{ maVariable1 | lowercase }}

{{ maVariable1 | titlecase }}

Nombres

TitleCasePipe

{{ "{{ value_expression | number [ : digitsInfo [ : locale ] ] }}" }}

minIntegerDigits . minFractionDigits - maxFractionDigits

Avant : {{ maVariable2 }}

{{ maVariable2 | number : '6.2-2' }}

{{ maVariable2 | number : '6.2-2' : 'en' }}
