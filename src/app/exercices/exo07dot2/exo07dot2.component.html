Exercice 07.2 - Convertisseur degré

Créer un pipe qui permet de convertir une temperature en degré celsius vers les degré fahrenheit et inversément.

temperature | toFahrenheit

temperature | toCelsius

temperature | temperature : 'c'

temperature | temperature : 'f'

{{ temperature }} °C : {{ temperature | toFahrenheit | number : '1.0-2' }}

{{ temperature }} °F : {{ temperature | toCelsius | number : '1.0-2' }}

{{ temperature }} °C : {{ temperature | temperature : 'f' | number : '1.0-2' }}

{{ temperature }} °F : {{ temperature | temperature : 'c' | number : '1.0-2' }}
